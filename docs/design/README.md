# Public Design Notes

This directory is reserved for sanitized public design summaries.

The public repository should not contain full internal Synapse design documents,
private fusion lists, monetization plans, local workflow notes, account
configuration, proxy settings, or personal memory material.

Current public design entry points:

- `docs/ARCHITECTURE_OVERVIEW.md`
- `docs/CAPABILITY_MATRIX.md`
- `docs/CLAIM_BOUNDARIES.md`
- `docs/PUBLIC_ROADMAP.md`
- `VERSIONING.md`

Internal design alignment is currently tracked as `Synapse Design V6.6`, but
internal `Vx.x` design numbers are not public software release numbers.
