# Architecture Overview

Synapse is a local-first personal cognitive kernel organized around one
governing core and three product centers.

## Taiheng / Governance Core

Taiheng coordinates mode, permissions, safety gates, recovery, audit, release
readiness, and cross-domain boundaries.

## Zhishu / Intelligence Hub

Zhishu manages knowledge, memory, rules, reusable skills, experience records,
and admission policy for durable knowledge.

## Xingtai / Action Desk

Xingtai manages directions, tasks, schedules, planning previews, opportunity
mining, project progress, and reviewable outputs.

## Baigong / Arsenal

Baigong manages tools, Agents, local automation, browser automation, data
source registration, information aggregation, and extension modules.

## Public Baseline Boundary

Synapse `0.0.0` exposes guarded local previews and local-first workflows. It
does not claim unrestricted automation, automatic external delivery, browser
write automation, cloud sync as a source of truth, or automatic durable
knowledge admission.
